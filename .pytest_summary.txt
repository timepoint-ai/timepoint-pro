================================================================================
QUICK START
================================================================================

1. Add markers to existing tests:
   python add_test_markers.py

2. Verify test collection:
   pytest --collect-only

3. Run tests by level:
   pytest -m unit              # Fast unit tests
   pytest -m integration       # Integration tests
   pytest -m system           # System tests
   pytest -m e2e              # E2E tests

4. Skip expensive tests:
   pytest --skip-slow         # Skip slow tests
   pytest --skip-llm          # Skip LLM tests

5. Run with real LLM:
   export OPENROUTER_API_KEY=your_key
   pytest --real-llm
